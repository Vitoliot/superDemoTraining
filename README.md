# Полезенейшие приложения для школы "Леарн" и магазина "Намордник"

## Функционал
1. просмотр услуг и товаров;
2. добавление услуг и товаров;
3. редактирование услуг и товаров;
4. сортировка, фильтрация и поиск по услугам и товарам;
5. режим администратора.
